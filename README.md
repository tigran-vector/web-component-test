## Installation

Run next command:

```bash
npm i
```

or

```bash
npm install
```

## Main command

**Run** project in development mode with watchers:

```bash
npm run start
```

**Build** project:

Development mode 
```bash
npm run build:dev
```

Production mode
```bash
npm run build:prod
```
